# Rule 001 發牌

使用 52 張撲克牌。

流程：

1. 洗牌
2. 切牌
3. 尾家看底牌
4. 發海底四張
5. 每家發牌：
   - 第一輪 1 張
   - 第二輪 2 張
   - 第三輪 3 張

共六張。

# Rule 003 有紅必吃紅

若可配對牌中包含紅牌：

♥ Heart
♦ Diamond

只能選擇紅牌。

若沒有任何紅牌：

♠ Spade
♣ Club

皆可自由選擇。

沒有花色優先順序。

---

# Rule 004 海底四張相同

若四張海底點數完全相同。

強制重發。

---

# Rule 005 玩家四張相同

玩家手牌 + 海底

若形成四張相同。

玩家可以要求重發。

---

# Rule 006 尾家四張相同

尾家：

手牌 + 海底 + 底牌

若形成四張相同。

且尾家手牌必須參與。

尾家可以要求重發。

---

# Rule 007 全黑

玩家六張手牌：

全部都是：

♠
♣

且不能出現：

A

9

即可要求重發。

---

# Rule 008 玩家回合

玩家出一張牌。

系統依規則找出可配對牌。

若存在紅牌。

只能選擇紅牌。

完成吃牌。

翻牌。

判斷是否連吃。

換下一位。
